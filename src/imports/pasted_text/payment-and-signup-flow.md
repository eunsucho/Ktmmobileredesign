주문완료 팝업과 셀프개통 완료 페이지까지 인터랙션 흐름을 모두 구현해줘.

[A. 요금 납부 방법 - 자동이체 vs 신용카드 분기]
============================================
"자동이체"와 "신용카드" 카드는 토글 라디오 버튼처럼 동작.
선택된 카드에 따라 아래 섹션이 완전히 다른 폼으로 전환됨 (조건부 렌더링).

──────────────────────────────────────────
▶ A-1) "자동이체" 선택 시 (기본값)
──────────────────────────────────────────
- 섹션 타이틀: "계좌정보"
- 가로 2분할 입력
  · 좌측: 은행명 셀렉트 (드롭다운 옵션: 국민은행, 기업은행, 농협, 아이엠뱅크, 부산은행, 산업은행, 새마을금고, 수협중앙, 신협, 신한은행, 우리은행, 우체국, 전북은행, 제주은행, SC제일은행, KEB하나은행, 시티은행, 케이뱅크, 카카오뱅크)
  · 우측: 계좌번호 입력 (placeholder "'-'없이 입력", 숫자만)
- 회색 안내 박스 (#F7F7F7 배경, 4px radius, padding 16~20px):
  · 납부방법은 본인명의의 카드 또는 계좌로만 등록 가능합니다.(미성년자의 경우 법정대리인 명의 포함)
  · 매월 21일 납부
  · 평생계좌(개인 핸드폰번호를 계좌번호로 사용하는 계좌)는 이용 불가능합니다.
- 버튼 라벨: "계좌번호 유효성 체크"

──────────────────────────────────────────
▶ A-2) "신용카드" 선택 시
──────────────────────────────────────────
- 섹션 타이틀: "신용카드 정보"
- 가로 100% 너비 입력 1개
  · 카드번호 입력 (플로팅 라벨 "카드번호", placeholder "'-'없이 입력", 숫자만)
- 그 아래 새 섹션 타이틀: "신용카드 유효기간"
- 가로 2분할 셀렉트
  · 좌측 셀렉트: 플로팅 라벨 "MM"
    - 드롭다운 옵션: 1월 ~ 12월
  · 우측 셀렉트: 플로팅 라벨 "YY"
    - 드롭다운 옵션: 2026년 ~ 2036년 (현재 연도부터 +10년)
  - 두 셀렉트 모두 좌측 ✓ 체크 마크는 없고 우측에 ∨ 화살표만
- 회색 안내 박스 (자동이체와 같은 스타일):
  · 납부방법은 본인명의의 카드 또는 계좌로만 등록 가능합니다.(미성년자의 경우 법정대리인 명의 포함)
  · 1회차 매월 10~11일경
- 버튼 라벨: "신용카드 유효성 체크"

  · 우측: "이벤트 보러가기" - 검정 배경(#000), 흰색 텍스트 bold, 너비 약 200px, 높이 56px

[E. 전체 인터랙션 흐름 정리]
============================================

1) 5단계 진입 → 자동이체 + 이메일 명세서 기본 선택, 주문완료 비활성
2) "신용카드" 카드 클릭 → 계좌정보 섹션이 신용카드 정보 + 유효기간 섹션으로 전환
3) 카드번호 입력 + MM/YY 셀렉트 선택
4) "신용카드 유효성 체크" 버튼 클릭 → 
   · 버튼 라벨이 "신용카드 체크 완료"로 바뀌고 흐려짐
   · 위쪽 입력 필드 모두 비활성화(회색 톤)
   · 하단 "주문완료" 버튼이 검정 활성 상태로 변경
5) "주문완료" 클릭 → "개통 후 주의사항 안내" 모달 출현 (오버레이)
6) 모달 본문 읽고 체크박스 클릭 → 초록 체크로 전환, "확인" 버튼 활성화
7) "확인" 클릭 → 모달 닫힘, 짧은 로딩 스피너 → 페이지 전환
8) "가입신청(상담사 연결)" 페이지 도착 → 초록 체크 + 완료 메시지 + 메인으로/이벤트 버튼

자동이체 흐름도 동일:
1)→ 은행 선택 + 계좌번호 입력 → "계좌번호 유효성 체크" 클릭 → 
"계좌번호 체크 완료"로 바뀌고 주문완료 활성화 → 이후 동일
